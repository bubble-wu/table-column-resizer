# macOS安装插件完整指南

## 方法1：显示隐藏文件（推荐）

### 步骤1：显示隐藏文件
1. 打开 **Finder**
2. 导航到你的 **Obsidian库文件夹**（包含你的.md文件的地方）
3. 按下快捷键：**Command + Shift + .**（点号）
4. 你会看到 `.obsidian` 文件夹出现了！

### 步骤2：创建插件目录
1. 双击打开 `.obsidian` 文件夹
2. 如果看到 `plugins` 文件夹就进去，没有就创建一个
3. 在 `plugins` 中创建 `table-column-resizer` 文件夹
4. 复制插件文件进去

### 步骤3：隐藏文件恢复（可选）
再按一次 **Command + Shift + .** 可以隐藏文件
