## Usage Instructions

1. **Switch to Preview Mode**: Press `Cmd+E` (Mac) or `Ctrl+E` (Windows/Linux)

2. **Hover over column borders**: Move your mouse to the right edge of any column header

3. **Look for the resize handle**: A red vertical line will appear when hovering

4. **Click and drag**: Click on the red handle and drag left/right to resize

5. **Release to set width**: Let go when the column is your desired width

## Tips for Screenshots

- **Default state**: Capture the table with default column widths
- **Hover effect**: Show the red resize handle appearing
- **Dragging**: Capture the moment of resizing with mouse cursor
- **Final result**: Show customized column widths
- **Settings panel**: Show the plugin configuration options
