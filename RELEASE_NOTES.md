# Table Column Resizer v1.0.0

## 🎉 初始稳定版本发布

Table Column Resizer 是一个 Obsidian 插件，允许用户通过拖拽来调整表格列宽，提升表格的可读性和美观度。

## ✨ 功能特性

- **拖拽调整列宽**：直观的鼠标拖拽操作
- **预览模式支持**：在预览模式下完美工作
- **持久化设置**：列宽设置自动保存
- **自定义限制**：可设置最小/最大列宽
- **视觉反馈**：清晰的悬停和拖拽指示器
- **主题兼容**：支持所有 Obsidian 主题
- **响应式设计**：适配不同屏幕尺寸

## 🔧 技术特点

- 使用 TypeScript 开发
- 遵循 Obsidian 插件开发规范
- 完整的错误处理和边界检查
- 优化的性能表现

## 🚀 安装使用

1. 在 Obsidian 设置中打开社区插件
2. 搜索 "Table Column Resizer"
3. 安装并启用插件
4. 在预览模式下拖拽表格列边框即可调整宽度
