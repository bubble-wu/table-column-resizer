# Table Column Resizer - 快速安装和测试指南

## 🚀 快速开始（5分钟完成）

### 步骤1：构建插件
```bash
# 确保你在插件目录中
cd /Users/bytedance/Desktop/Code\ Base/table-column-resizer

# 构建插件
npm run build

# 确认构建成功（应该没有错误）
```

### 步骤2：安装到Obsidian

1. **打开Obsidian**，进入你的库
2. **打开插件文件夹**：
   - Windows: `设置 → 社区插件 → 浏览旁边的文件夹图标`
   - Mac: `Settings → Community Plugins → folder icon next to Browse`
3. **创建插件目录**：
   - 在打开的文件夹中，创建新文件夹：`table-column-resizer`
4. **复制文件**：
   - 复制 `main.js` → 插件文件夹
   - 复制 `manifest.json` → 插件文件夹  
   - 复制 `styles.css` → 插件文件夹
5. **启用插件**：
   - 返回Obsidian，关闭"安全模式"
   - 在插件列表中找到"Table Column Resizer"
   - 点击启用

### 步骤4：测试功能

1. **切换到预览模式**：
   - 点击右上角的"预览"按钮，或
   - 按 `Ctrl/Cmd + E`

2. **测试拖拽功能**：
   - 将鼠标移动到表格列的**右边缘**
   - 看到**半透明拖拽手柄**出现
   - **按住鼠标左键**拖拽调整列宽
   - **释放鼠标**完成调整

3. **验证效果**：
   - ✅ 列宽应该实时跟随鼠标移动
   - ✅ 拖拽时表格应该有蓝色边框高亮
   - ✅ 最小列宽限制为50px
   - ✅ 最大列宽限制为500px

## 🐛 常见问题快速解决

### ❌ 拖拽手柄不显示
**检查**：
1. 是否在**预览模式**？（不是编辑模式）
2. `styles.css`是否复制到插件文件夹？
3. 插件是否已**启用**？

### ❌ 列宽无法调整
**检查**：
1. 是否将鼠标放在**列的右边缘**？
2. 控制台是否有错误？（Ctrl/Cmd + Shift + I）
3. 重启Obsidian试试

### ❌ 设置不生效
**检查**：
1. 修改设置后是否**等待自动保存**？
2. 重新打开文档测试
